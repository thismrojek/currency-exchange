# Currency exchange
Exchanging currencies has never been so easy!  
A simple application that allows you to check the exchange rate.

## Development
To run development server for this project run
```
  npm run dev
```

## DOTENV template
Here is an .env template, that you can actually use to use required REST API services:
```
CURRENCY_API_ENDPOINT=
CURRENCY_API_KEY=
```
